# Туристический сайт на WordPress

Проект: туристический сайт с подбором туров, авиабилетов и проживания.

## ?? Используемые технологии
- WordPress + тема OceanWP
- Конструктор Elementor (бесплатный)
- HTML-формы без плагинов
- Адаптивная вёрстка

## ?? Страницы
- Главная
- Тур-пакеты
- Авиабилеты
- Проживание
- Новости (блог)
- Контакты
- О нас
- Забронировать

## ?? Скриншоты
Смотри папку `/screenshots/`

## ?? Для локального запуска:
Установи WordPress локально (например, через OSPanel) и помести папку в директорию `/home`.
